# Philippines 2026 Digital Payments & E-Commerce Dashboard

A beginner-friendly data analytics portfolio project built with Excel and Power BI. This project explores how Filipinos pay and shop in 2026 using publicly available data on digital payments, e-commerce, and consumer behavior.

## Project goal

The goal of this project is to analyze digital payment growth and online shopping trends in the Philippines in 2026, then present insights through a clean dashboard and short business case study.

## Business questions

- How fast are digital payments growing in the Philippines?
- How close is the Philippines to becoming more cashless?
- How are Filipinos spending online in 2026?
- What does consumer behavior say about trust, value, and convenience?
- What insights can businesses, banks, and fintech companies learn from these trends?

## Tools used

- Microsoft Excel
- Power BI
- GitHub
- Public Philippine and industry data sources

## Dataset structure

This project uses a small manually curated dataset for beginner-friendly analysis.

### Sheets

- `payments_trend`
- `consumer_behavior`
- `macro_context`

## Planned workflow

1. Define business questions
2. Collect and encode verified data in Excel
3. Clean and structure the dataset
4. Build a Power BI dashboard
5. Write key insights and recommendations
6. Publish the project on GitHub
